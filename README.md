# TextSense

LSTM vs BERT on IMDb movie reviews.

Result: BERT reached 92% F1 score.

Run:
git clone https://github.com/tinkywinkygluestick799-dotcom/textsense
cd textsense
pip install -r requirements.txt
python train_bert.py

Dependencies: Python 3.8+, Transformers, PyTorch, scikit-learn
